# Flagstaff Water Futures

Complete Docusaurus working site prepared for Flagstaff Water Group review.

## Run locally

1. Install Node.js 20 or later.
2. Open a terminal in this folder.
3. Run `npm install`.
4. Run `npm start`.
5. Open the address shown in the terminal (normally http://localhost:3000).

## Build for publication

Run `npm run build`. The deployable static website will be created in `build/`.

## Edit content

- Front-facing pages: `src/pages/`
- Water Library pages: `docs/`
- Colors and layout: `src/css/custom.css`
- Top navigation: `docusaurus.config.js`

This is a working draft. Verify project costs, dates, regulatory claims, and scientific statements before public release.
