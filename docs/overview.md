# Water Library

This library supports the front-page case for a full evaluation of Advanced Water Treatment. It separates background facts, project descriptions, comparison questions, uncertainties, and sources.

## How to use it

- Start with **Flagstaff Today** for the current system and future need.
- Use **The Alternatives** for concise descriptions of AWT and RGR.
- Use **Key Issues** for water quality, funding, technology adoption, and risk.
- Use **Project Resources** for the timeline, sources, and open questions.

> This is a working knowledge base. Statements based on FWG analysis should not be confused with official City findings.
